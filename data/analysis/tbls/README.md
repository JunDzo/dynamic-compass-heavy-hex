# Breaker Tables

The `.txt` files in this folder are the measurement schedules used in the paper. They are generated for a maximum code distance of `d = 35`.

For Fig. 11 in the paper:

- `breaker_table_d35_0.txt`: schedule A.
- `breaker_table_d35.txt`: schedule B.
- `breaker_table_d35_2.txt`: schedule C.

Helper scripts:

- `select_breakers_interactive.py`: opens an interactive grid for selecting breaker positions and writing a breaker table.
- `visualize_breaker_table.py`: renders a saved breaker table so the schedule pattern can be inspected visually.
