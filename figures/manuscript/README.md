# Manuscript Figures

This folder contains figure assets and supporting stats used for the manuscript.

- `no-reset-*`: no-reset threshold plots and stats used during manuscript figure preparation.
- `no-reset-finer`: finer no-reset threshold slices and stats.
- `no-reset-si1000`: no-reset threshold slices generated with the SI1000-noise-model setting.
- `various-d`: logical-error-rate comparison across distances.

Rotation videos are currently stored in `figures_manuscript/`; they can be moved here as `reset/` and `no-reset/` if the manuscript assets are consolidated further.
